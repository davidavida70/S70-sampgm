# Sobre a gamemode
Fiz esta gamemode como uma prática de código, porém ao longo do tempo me apeguei e adicionei alguns itens para ficar divertido jogar com amigos(as). Atributos:
- Integração com banco de dados MariaDB que armazena por exemplo coordenadas de veículos, ID da skin, quantidade de dinheiro, etc;
- Cargos (motorista de ônibus, policial, detento, desempregado);
- Compra e venda de veículos (lista enxuta mas facilmente alterável);
- Prisão e soltura de criminosos;
- Sistema de senha para nomes de contas;
- Filterscripts (arquivos adicionais) para mais comandos e funções.

## Como usar a minha gamemode no seu servidor
- Baixe o código base e substitua tudo na pasta SAMP.

# O que é SA-MP?
- SA-MP é uma criação da comunidade, um multijogador online feito para uso em conjunto com o jogo Grand Theft Auto: San Andreas, permitindo que jogadores interajam com quase tudo o que o jogo base tem para oferecer. Existem versões para Windows, Linux e Android.

![samp](https://github.com/user-attachments/assets/7c3acdf6-903e-4ae6-9e4e-2f72dccf5589)

## Como faço pra jogar?
- Siga as instruções deste site, com o GTA:SA já instalado: https://www.sa-mp.mp/downloads/
- Para android, busque na Play Store por "samp launcher" e baixe o que lhe agradar mais.
- Escolha um servidor na lista e jogue!
